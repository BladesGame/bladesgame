# 騎砍2 | 領主忠誠與叛變預防：穩住你的權力基礎

---

## 前言

領主叛變是王國崩塌的第一大原因。你辛苦打下十座城，三個領主帶著封地跑掉，瞬間回到解放前。這篇教你怎麼避免。

---

## 一、為什麼領主會叛變

| 原因 | 嚴重度 |
|------|:--:|
| 沒有封地或封地太少 | 🔴 最高 |
| 關係值過低 | 🔴 高 |
| 你一直否決他的提案 | 🟡 中 |
| 你的王國戰敗連連 | 🟡 中 |
| 有其他國王挖角他 | 🟡 中 |
| 文化不同（他帝國人你庫塞特） | 🟢 低 |

---

## 二、關係值管理

### 2.1 加關係的方法

| 方法 | 效果量 |
|------|:--:|
| 分封地給他們 | +++++ |
| 支持他們的政策提案 | +++ |
| 一起打仗 | ++ |
| 投票時支持他們 | + |
| 舉辦宴會（需要特定條件） | + |

### 2.2 扣關係的行為

| 行為 | 效果量 |
|------|:--:|
| 把別人提名給他的封地搶走 | ----- |
| 否決他的政策提案 | --- |
| 投票反對他 | -- |
| 燒他原本所屬國的村莊 | - |

---

## 三、封地策略

### 3.1 黃金比例

| 領主類型 | 建議封地數 |
|------|:--:|
| 剛挖來的 | 1 座（立刻給） |
| 核心支持者 | 2-3 座 |
| 一般領主 | 1-2 座 |
| 國王自己 | 2-3 座（首都+經濟城） |

> 單一領主超過 3 座城就很危險，他會覺得自己比你強。

### 3.2 分配節奏

不要一次給完。每打下一兩座城給一個領主，讓他們有「跟著國王有好處」的預期。

---

## 四、預防挖角

其他國王會挖你的領主，尤其是：
- 文化跟對方國王相同的領主
- 沒封地或封地少的領主
- 對你不滿的領主

### 防挖角對策

1. 確保每個領主都有封地（最基本）
2. 關係值維持正數
3. 不讓領主跟敵國領主有太多接觸（別派去邊境）
4. 裝 Diplomacy MOD → 可以看領主被挖角的風險

---

## 五、叛變後的處理

萬一還是叛變了：

### 5.1 立即行動

1. 叛變領主帶著你的封地加入敵國
2. **那塊封地現在是敵國領土**
3. 立刻宣戰敵國 → 把那座城打回來
4. 俘虜叛變領主 → 處決或永久監禁

### 5.2 殺一儆百？

處決叛變領主有威懾效果，但也會大幅扣你跟所有領主的關係。除非他有太多封地造成巨大損失，否則不建議處決。

---

## 六、穩定王國的長遠策略

1. **文化統一**：優先挖角跟你同文化的領主
2. **封地不要一次給太多**：讓他們一直有追求
3. **戰事順利**：一直打勝仗的國王不會有人叛變
4. **政策保護**：元老院政策 + 皇室衛隊
