# 騎砍2 | 政策與封地管理：王國不是打完就沒事

---

## 前言

打下城很爽，但管城很煩。每個城要總督、要駐軍、要蓋建築、要顧繁榮度。這篇把城鎮管理的眉角講清楚。

---

## 一、城鎮管理面板

點你的城鎮 → 管理城鎮：

| 指標 | 影響 |
|------|------|
| 繁榮度 | 稅收基礎，越高錢越多 |
| 忠誠度 | 低於 25 會叛變 |
| 糧食 | 不夠會飢餓，繁榮度下降 |
| 安全感 | 低會出盜賊 |
| 駐軍 | 防守力量，但要花錢養 |

---

## 二、建築優先級

每個城可以蓋建築，但欄位有限：

| 優先級 | 建築 | 效果 |
|:--:|------|------|
| 1 | 穀倉 | 糧食儲備上限↑，圍城撐更久 |
| 2 | 果園 | 糧食產量↑，支撐更高繁榮度 |
| 3 | 市場 | 稅收↑ |
| 4 | 城牆 | 防禦力↑ |
| 5 | 軍營 | 駐軍上限↑ |

> 永遠先蓋穀倉和果園。糧食是一切繁榮的基礎。

---

## 三、總督指派

### 3.1 為什麼要總督

把一個夥伴/家族成員設為城鎮總督：
- 總督的文化跟城鎮相同 → 忠誠度 +1
- 總督的技能加成（管理、工程等）會套用在城鎮
- 裝 Governor Handle Issues MOD → 總督自動處理城鎮隨機事件

### 3.2 選誰當總督

| 優先看 | 原因 |
|------|------|
| 文化 | 跟城鎮文化相同最重要 |
| 管理學 | 影響稅收和建設速度 |
| 工程學 | 影響建築速度 |

---

## 四、村莊管理

每個城鎮有 2-4 個附屬村莊：

- 村莊的產出決定城鎮的原料供應
- 村莊被掠奪 → 原料停產 → 城鎮經濟崩
- 裝 Improved Garrisons MOD → 可以派巡邏隊保護村莊

---

## 五、政策推薦

當你有 5+ 個城鎮時，這些政策特別有價值：

| 政策 | 效果 | 適合情境 |
|------|------|------|
| 土地稅 | 城鎮稅收 +5% | 城多時收入可觀 |
| 公民權 | 城鎮忠誠度 +1 | 征服不同文化城鎮時 |
| 道路收費 | 村莊收入 +3% | 村莊多的王國 |
