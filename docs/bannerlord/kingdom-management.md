# 騎砍2 | 王國管理深度指南：當國王比打仗難

---

## 前言

很多人以為打下城、建國就破關了。實際上，**當國王比當傭兵難十倍**。你要管領主心情、封地分配、政策投票、外交關係、經濟收支……一個沒弄好，王國就崩了。

---

## 一、王國界面導覽

打開王國介面（K鍵），你會看到：

| 分頁 | 內容 |
|------|------|
| 王國 | 王國總覽、外交狀態 |
| 封地 | 所有城鎮和城堡列表 |
| 領主 | 所有領主、他們的封地、關係值 |
| 政策 | 已頒布和可頒布的政策 |
| 軍團 | 當前活躍的軍團 |

---

## 二、封地分配

### 2.1 分配原則

這是最容易搞砸的環節。領主沒封地 = 不滿 = 叛變。

| 原則 | 說明 |
|------|------|
| 新領主先給一座 | 剛挖來的領主第一座城最重要 |
| 不要讓一人坐大 | 單一領主超過 3 座城 = 危險 |
| 自己保留核心城 | 首都 + 1-2 座經濟城自己留 |
| 邊境城給強領主 | 邊境容易被打，給兵多的領主 |

### 2.2 分配投票

打下新城 → 王國投票決定歸誰。

- 你可以提名任何人（包括自己）
- 你的一票權重最高（你是國王！）
- 領主投票看關係值和他們現有封地數量

---

## 三、政策系統

### 3.1 推薦政策

| 政策 | 效果 | 推薦度 |
|------|------|:--:|
| 皇室衛隊 | 國王部隊上限 +60 | ⭐⭐⭐ |
| 土地稅 | 城鎮稅收 +5% | ⭐⭐⭐ |
| 元老院 | 領主忠誠度 +1 | ⭐⭐ |
| 公民權 | 城鎮忠誠度 +1 | ⭐⭐ |

### 3.2 避免的政策

| 政策 | 原因 |
|------|------|
| 農奴制 | 領主扣關係，民眾忠誠降 |
| 狩獵權 | 好處有限，領主不一定支持 |

> 每次提政策前先看支持率。低於 50% 的不要硬推，會被否決還扣關係。

---

## 四、領主關係管理

### 4.1 關係值影響什麼

- 關係高 → 投票支持你 → 帶兵跟你打仗
- 關係低 → 投票反對你 → 可能叛變

### 4.2 怎麼提升關係

| 方法 | 效果 |
|------|------|
| 分封地給他們 | 大幅提升 |
| 支持他們的政策提案 | 小幅提升 |
| 一起打仗 | 慢慢累積 |
| 送禮物（Mod） | 裝 Diplomacy MOD 後可用 |

### 4.3 叛變徵兆

- 關係值持續下降
- 封地太少卻一直要求更多
- 經常投票反對你
- 不帶兵參加軍團

> 發現這些徵兆 → 立刻給封地或提升關係，不要等到叛變。

---

## 五、經濟管理

### 5.1 王國收入

| 來源 | 說明 |
|------|------|
| 個人封地稅收 | 你直轄的城鎮 |
| 工廠/商隊 | 個人被動收入 |
| 戰爭賠款 | 停戰時對方付的錢 |
| 貢金 | 附庸國付的錢 |

### 5.2 王國支出

| 支出 | 說明 |
|------|------|
| 部隊薪水 | 最大的支出 |
| 駐軍薪水 | 城鎮/城堡的守軍 |
| 貢金 | 你付給其他國家的 |

> 💡 永遠確保日收入 > 日支出。赤字持續太久 = 兵跑光。

---

## 六、進階管理技巧

### 6.1 集中力量打一國

不要同時跟多國開戰。一次打一個，打完停戰再打下一個。

### 6.2 用軍團集結

創建軍團（花影響力）→ 呼叫領主帶兵加入 → 集中兵力攻城。軍團戰力遠大於各自為戰。

### 6.3 收附庸而非占領

有些小國家可以威脅附庸而不需要完全征服。附庸國會付貢金，不佔你的封地。
