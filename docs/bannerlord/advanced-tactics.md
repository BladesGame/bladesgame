# 騎砍2 | 混合兵種進階戰術：不是人多就會贏

---

## 前言

90% 的戰鬥輸掉，不是因為兵不夠多，是因為兵種搭配錯了。這篇把進階戰術思維講清楚——從陣型邏輯到針對性克制。

---

## 一、兵種三角克制

```
        步兵（扛線/反騎）
         /          \
        /            \
  弓兵（遠程輸出）← 被騎兵剋 → 騎兵（衝鋒/機動）
        \            /
         \          /
          互相配合
```

- 騎兵衝鋒 → 剋弓兵（弓兵被近身就完了）
- 步兵盾牆 → 剋騎兵（騎兵衝不穿密集步兵）
- 弓兵遠程 → 剋步兵（步兵走過來先被射殘）

---

## 二、陣型邏輯

### 2.1 層次布陣

```
[敵方]
  ↓
─── 弓兵（最後排，高地）───  ← 最安全，輸出最大化
─── 步兵（中排，盾牆）───     ← 保護弓兵，扛敵方衝鋒
─── 騎兵（兩翼）───           ← 機動打擊，繞背衝鋒
```

### 2.2 為什麼要分層

- 弓兵在最後 → 不被敵方騎兵衝
- 步兵在中間 → 什麼都能扛
- 騎兵在兩翼 → 隨時出擊或保護側面

---

## 三、針對性克制戰術

### 3.1 敵方是騎兵主力（庫塞特/瓦蘭迪亞）

```
對策：
- 步兵排方陣（F3+F3）扛騎兵衝鋒
- 弓兵射馬（馬倒了騎兵就是步兵）
- 地形選森林或橋樑
- 不要追他們（你追不上）
```

### 3.2 敵方是弓兵主力（巴坦尼亞）

```
對策：
- 騎兵直接衝他們弓兵群（弓兵近戰 = 死）
- 步兵舉盾推進（盾牌擋箭）
- 不要在高地跟他們對射（你射不贏費奧納勇士）
```

### 3.3 敵方是步兵主力（斯特吉亞/帝國）

```
對策：
- 弓兵放高處射，消耗他們
- 騎兵反覆衝鋒（衝完拉走，不要留在裡面）
- 你自己的步兵不要正面對撞（斯特吉亞斧兵近戰無敵）
```

---

## 四、兵種配比黃金法則

| 軍隊規模 | 步兵 | 弓兵 | 騎兵 |
|------|:--:|:--:|:--:|
| 小隊（50 人以下） | 40% | 30% | 30% |
| 中隊（50-100 人） | 35% | 35% | 30% |
| 大軍（100+ 人） | 30% | 35% | 35% |

> 永遠不要 100% 同一種兵。單一兵種 = 被剋到死。

---

## 五、戰鬥指揮 check list

每場戰鬥開始時：

- [ ] 弓兵找到高地了嗎？
- [ ] 步兵開盾牆了嗎？
- [ ] 騎兵在側翼了嗎？
- [ ] 看清楚敵方兵種組成？（騎兵多？弓兵多？）
- [ ] 地形對誰有利？（森林=騎兵廢，高地=弓兵強）
