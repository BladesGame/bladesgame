# 騎砍2 | 遊戲設定與效能優化：不卡頓的卡拉迪亞

---

## 前言

騎砍2的千人戰場很吃效能。這篇幫你調到最佳平衡——畫面不算太醜、打仗不會卡。

---

## 一、系統需求

| | 最低 | 建議 |
|------|------|------|
| CPU | i3-8100 / Ryzen 3 1200 | i5-9600K / Ryzen 5 3600 |
| RAM | 8 GB | 16 GB |
| GPU | GTX 660 / HD 7850 | GTX 1060 / RX 580 |
| SSD | 建議 | 強烈建議（加快讀取） |

> 這遊戲吃 CPU > GPU。千人戰場卡頓通常是 CPU 瓶頸。

---

## 二、關鍵設定

### 影響效能最大的設定

| 設定 | 建議 | 原因 |
|------|:--:|------|
| 戰場人數 | 500-800 | 超過 1000 大部分 CPU 撐不住 |
| 陰影品質 | 中或低 | 最吃效能的設定 |
| 植被細節 | 中 | 野外戰鬥畫面元素多 |
| 粒子效果 | 低 | 攻城戰煙霧和火焰 |
| 解析度縮放 | 90-100% | 降到 90% 幾乎看不出差異 |

### 可以開高的設定

- 紋理品質（看 VRAM，4GB+ 可以開高）
- 角色細節
- 地形品質

---

## 三、音效設定

| 設定 | 建議 |
|------|:--:|
| 音效頻道數 | 128（預設 256 吃 CPU） |
| 音樂 | 開（不影響效能） |

---

## 四、進階優化

### 4.1 啟動選項

Steam → 騎砍2 → 內容 → 啟動選項：

```
-high -malloc=system
```

### 4.2 Windows 設定

- 遊戲模式：開啟
- 電源計畫：高效能
- 關閉背景錄影（Xbox Game Bar）

### 4.3 NVIDIA 控制面板

- 電源管理模式 → 最大效能
- 垂直同步 → 關閉（減少輸入延遲）
