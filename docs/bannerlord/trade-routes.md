# 騎砍2 | 終極跑商路線圖：從行商到商業帝國

---

## 前言

在卡拉迪亞，打仗很帥，但跑商穩定。一條好路線跑順了，日入上萬不是問題，而且**零風險**——不用怕兵死光、不用怕被俘虜。

---

## 一、跑商基礎

### 1.1 物價怎麼看

滑鼠移到商品上 → **綠色** = 比均價便宜 → 買進。**紅色** = 比均價貴 → 賣出。

### 1.2 影響物價的因素

- 城鎮周圍村莊生產什麼 → 原料便宜
- 城鎮缺什麼 → 需求品貴
- 商隊來往頻率 → 頻繁進出會拉平物價
- 戰爭狀態 → 被圍城/掠奪過的城物價飆漲

### 1.3 必備裝備

- **騾子/駱駝 5-8 頭**：增加負重，太多拖慢移速
- **交易 perk**：批發商（降低貿易懲罰）、市場經紀人（自動更新物價）

---

## 二、金幣路線

### 🥇 穀物線（最穩）

| 起點 | 終點 | 利潤/趟 |
|------|------|:--:|
| 南帝國村莊（達努斯蒂卡 Danustica 周邊） | 北帝國城鎮 | 3000-5000 |
| 庫塞特東部村莊（奧多赫 Odokh 周邊） | 帝國城鎮 | 2000-4000 |

> 南帝國達努斯蒂卡周圍村莊盛產穀物：**Menzura、Qasira、Asmait** 等村，穀物價格全大陸最低。

> 穀物永遠有需求，戰爭期間糧食價格翻倍。

### 🥈 硬木線

| 起點 | 終點 | 利潤/趟 |
|------|------|:--:|
| 巴坦尼亞村莊 | 阿塞萊城鎮 | 2000-4000 |
| 巴坦尼亞村莊 | 斯特吉亞城鎮 | 1500-3000 |

> 🪓 巴坦尼亞森林多，硬木便宜到哭。重點村莊：**Glenlithrig、Tor Leiad、Mag Arba、Ebbseth、Tor Melina** — 這些村子圍繞在 Marunath、Seonon、Dunglanys 周圍，硬木價格常年在 15-30 金。

### 🥉 牲畜線

| 起點 | 終點 | 利潤/趟 |
|------|------|:--:|
| 庫塞特東部村莊（Balkhans、Odokh 周邊） | 帝國城鎮 | 2000-3000 |
| 阿塞萊沙漠村莊（Husn Fulq、Razih 周邊） | 瓦蘭迪亞 | 2000-3500 |

> 牲畜會自己走路（不用負重），但移動速度會變慢。

### 4. 銀礦/珠寶線

| 起點 | 終點 | 利潤/趟 |
|------|------|:--:|
| 斯特吉亞銀礦村（Omor、Balgard 周邊） | 帝國/瓦蘭迪亞 | 3000-6000 |

> 斯特吉亞 Omor 和 Balgard 周邊有全大陸唯一的銀礦村莊，珠寶原料產地。

> 利潤最高，但需求量較小，需要等城鎮消化。

---

## 三、進階技巧

### 3.1 貿易傳聞

進城 → 點交易 → 看周圍城鎮價格浮動。這功能可以讓你不用親自跑過去就知道哪邊便宜哪邊貴。

### 3.2 戰時發財

打仗期間：
- 被圍城的城鎮糧食價格暴漲 → 運糧食去賣
- 被掠奪的村莊原料停產 → 相關商品會漲價
- 敵對國城鎮無法貿易 → 中立國城鎮商機大增

### 3.3 工廠聯動

在跑商路線的起點/終點買工廠，形成閉環：

```
起點買原料（便宜）→ 工廠加工 → 終點賣成品（貴）
```

例如：在巴坦尼亞買木材廠，原料自己產，成品賣到阿塞萊。

---

## 四、完整跑商環線

### 推薦環線（一圈約 5-7 天）

```
南帝國達努斯蒂卡 Danustica（買穀物，村：Menzura、Qasira）
  → 北帝國阿爾戈隆 Argoron（賣穀物）
  → 巴坦尼亞馬魯納斯 Marunath（買硬木，村：Glenlithrig、Tor Leiad）
  → 阿塞萊胡森富爾 Husn Fulq（賣硬木）
  → 庫塞特奧多赫 Odokh（買牲畜，村：Balkhans）
  → 帝國（賣牲畜）
  → 回到起點（重複）
```

> 一圈淨利約 8000-15000 金，跑兩圈就能開一間工廠。

---

## 結語

跑商的秘訣不是找到一條神路線，而是建立**環線**——讓每一趟都有貨可帶，不空車跑路。
