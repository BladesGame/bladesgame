# 騎砍2 | 鐵人模式生存指南：一條命通關卡拉迪亞

---

## 前言

鐵人模式 = 不能手動存檔 + 人物死了就沒了。這不是給新手玩的。但如果你覺得普通模式太簡單，這篇教你怎麼活下去。

---

## 一、鐵人模式規則

- 只有一個存檔，自動儲存
- 無法手動存檔讀檔（SL大法無效）
- 你在戰鬥中死亡 → 可以選擇繼續扮演繼承人，或遊戲結束
- 每個決定都是永久的

---

## 二、心態調整

| 普通模式 | 鐵人模式 |
|------|------|
| 輸了讀檔 | 輸了就輸了 |
| 隨便衝鋒 | 每一場戰鬥都要計算 |
| 俘虜隨便殺 | 俘虜是重要資產 |
| 錢花光沒差 | 錢花光 = 死 |

---

## 三、生存法則

### 3.1 永遠不要自己衝第一個

普通模式你可以騎馬衝進敵陣耍帥。鐵人模式你被砍下馬 = 死。讓小兵先上。

### 3.2 保守接戰

- 敵方兵力 > 你的 80%：不要打
- 敵方有大量騎兵而你步兵不夠：不要打
- 地形對你不利：不要打
- 不確定能不能贏：不要打

### 3.3 留退路

- 永遠留至少養兵 14 天的錢
- 永遠留一條逃跑路線（Tab 撤退）
- 被俘之前先跑，不要賭

### 3.4 醫療優先

- 找醫者型夥伴當外科醫生
- 醫學技能直接影響戰鬥後多少人能活下來
- 傷兵恢復速度快 = 你的部隊永遠滿編

---

## 四、鐵人模式專屬策略

### 4.1 龜到強為止

第一年不要加入任何國家。專心貿易賺錢、練兵等、開工廠。等你家族等級 3、有 100+ 精兵、50 萬存款，再考慮參戰。

### 4.2 選安全的出身

- 文化：瓦蘭迪亞（背靠海，不易被夾攻）
- 出身：商人（起始錢多，不用冒險打劫匪）

### 4.3 隨時準備接班人

- 盡早結婚生子
- 確保孩子成年時你還沒死
- 家族成員多 = 死了一個還有備胎

---

## 五、鐵人模式推薦 MOD

| MOD | 原因 |
|------|------|
| Diplomacy | 避免不必要的戰爭 |
| Improved Garrisons | 自動防禦，不用你親自守城 |
| RTS Camera | 死了還能指揮部隊 |
