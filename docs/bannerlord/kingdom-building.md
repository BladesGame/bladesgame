# 騎砍2 | 自立為王完全攻略：從流浪漢到一國之君

---

## 前言：想當國王？先問自己這三件事

騎砍2最終極的目標就是——**建立你自己的王國**。

但在你腦中浮現「我要稱霸卡拉迪亞」的畫面時，先冷靜一下，問自己：

1. 我的家族等級到 3 了嗎？
2. 我手下有至少 150 個精銳士兵嗎？
3. 我口袋裡有至少 **100 萬第納爾** 嗎？

如果以上有任一個答案是「沒有」，先把這篇存起來，回去練功。

如果都有了——恭喜你，可以開始看這篇了。

---

## 一、路線圖：從雇傭兵到國王

```
雇傭兵 → 封臣 → 累積實力 → 脫離國家 → 宣布獨立 → 打下第一座城 → 建立王國
  ↑                                                            ↓
  └──────────── 被其他國家宣戰 ← 守住領土 ← 招募領主 ←────────┘
```

## 二、第一階段：當封臣學經驗（家族等級 2-4）

### 為什麼不直接獨立？

因為你對這遊戲的國戰機制一無所知。

**先加入一個國家當封臣**，你會學到：

- 軍團怎麼召集
- 打仗勝負的影響
- 外交和投票系統
- 攻城需要多少準備
- 哪些領主好相處（之後可以挖角）

### 選哪個國家當封臣？

| 國家 | 適合原因 |
|------|------|
| 帝國（任一） | 戰爭最多，實戰經驗值滿滿 |
| 瓦蘭迪亞 | 地圖角落，比較不會被多線夾攻 |
| 庫賽特 | 跟著強國混，勝率高 |

> 💡 建議選**國力中等**的國家。太強的你脫離後難打，太弱的可能還沒學到東西國家就亡了。

### 在封臣期間該做什麼？

- **打仗**：累積聲望，提升家族等級
- **存錢**：目標 100-200 萬第納爾
- **練兵**：把主力部隊練到頂級
- **觀察**：記下哪些領主對國王不滿（之後可以挖角）
- **收集情報**：了解各城市的守軍實力

但在這之前，有一件事比當封臣更重要——

### ⚡ 前置策略：用雇傭兵身分削弱強國

在你正式當封臣之前，**強烈建議先用雇傭兵的身分在大陸上攪局**。

邏輯很簡單：

1. 你未來要自立，最大的威脅是什麼？不是某個國王討厭你，是**某個國家已經統一了半張地圖**
2. 雇傭兵的最大優勢是**不會跟任何人變成敵人**——你加入A國打B國，退出後B國不會恨你
3. 所以你可以**反覆橫跳**：今天幫帝國打庫塞特，明天幫庫塞特打瓦蘭迪亞，後天幫瓦蘭迪亞打帝國
4. 你的目標不是幫誰打贏，是**確保沒有任何一國能獨大**

> 🧠 **一句話：在自立之前，用雇傭兵身分當全大陸的「平衡器」。誰太強你就去打誰，打完就跳去另一邊。**

這樣等你宣布獨立時，面對的是一個各國互相制衡的卡拉迪亞，而不是某個已經坐擁30座城的怪物。

關於雇傭兵戰略的更多細節，詳見 [騎砍2 | 新手求生指南](./beginner-guide)。

## 三、第二階段：獨立宣言（關鍵時刻！）

### 時機判斷 ✅

以下條件**至少滿足四個**再考慮獨立：

- [ ] 家族等級 4 以上
- [ ] 資金 100 萬第納爾以上
- [ ] 精銳部隊 150+（不要用新兵湊數）
- [ ] 至少有一位夥伴可以當領主
- [ ] 你所在國家正在多線作戰（沒空理你）
- [ ] 選定的第一個攻城目標防禦薄弱

### 脫離國家的方式

**方式 A：和平分手（推薦）**

去王國介面，選擇「放棄封臣身分」→ 保留封地。

- ✅ 不會立即開戰
- ❌ 你會失去原有封地

**方式 B：帶封地叛逃（高風險）**

你身上有封地，直接跟國王說掰掰，把封地帶著走。

- ✅ 直接有領地可以經營
- ❌ 原國家會**立即對你宣戰**

> ⚠️ **新手千萬別選 B**，你會被碾過去。先和平分手，自己去打一座城比較安全。

---

## 四、第三階段：打下第一座城（最難的一步）

### 選對目標很重要

**好的攻城目標：**
- 防守兵力少於 200 人
- 糧食儲備低（圍城時間短）
- 附近沒有敵方大軍

**爛的攻城目標：**
- 任何帝國內陸的大城（守軍多、援軍快）
- 跟多國接壤的城（一打下就被圍毆）

### 攻城流程

1. **準備部隊**：150+ 精兵，帶攻城器械
2. **包圍城市**：圍城期間敵方糧食會消耗，守軍會減少
3. **建造器械**：建造投石機、攻城塔
4. **總攻**：器械建好後發起進攻
5. **拿下！**：恭喜，你有第一座城了！

> 💡 圍城時注意地圖，如果有敵軍大部隊靠近，寧可先撤退也不要硬打。

---

## 五、第四階段：建立王國

### 設立王國

佔領至少一座城/城堡後，打開家族介面 → 點「建立王國」。

系統會讓你：
- 取國名（慎重，不能改）
- 選旗幟（慎重，不能改）
- 宣布獨立

從這一刻起，你就是正式的國王了！

### 立國後第一件大事：活下來

新成立的王國 = **全大陸的肥羊**。

**生存策略：**

1. **先防守，再擴張**：不要急著打第二座城
2. **招募雇傭兵家族**：去酒館找小陣營領袖，付錢讓他們幫你打仗
3. **挖角敵方領主**：找對原國王不滿的領主，用錢說服他們加入你（超貴，但必須這麼做）
4. **求和**：花錢跟交戰國停戰，一次只打一國

---

## 六、第五階段：管理你的王國

### 招募領主

- 跟領主對話 → 說服他們加入
- 需要高魅力值（或很多很多錢）
- 條件：領主必須對原國王不滿

### 頒布政策

| 政策 | 效果 |
|------|------|
| 元老院 | 增加領主忠誠度 |
| 土地稅 | 增加稅收 |
| 皇室衛隊 | 增加你個人的部隊上限 |

> 政策有好有壞，頒布前看清楚。

### 分配封地

打下的城要分給領主，不然他們會不滿。

- 第一座城自己留著
- 後續的新城分給新加入的領主
- 避免同一個領主有太多封地（太強會造反）

## 🕳️ 自立常見失敗原因

| 失敗原因 | 解法 |
|------|------|
| 太早獨立，被聯軍輾過 | 存夠 100 萬和 150+ 精兵再獨立 |
| 沒領主幫忙，一人扛全線 | 獨立前就鎖定好要挖角的領主 |
| 經濟崩潰，兵跑光 | 確保工廠和商隊都在運作 |
| 打錯第一座城 | 選邊境小城，避開核心區 |
| 同時跟多國開戰 | 花錢求和，一次只打一國 |

---

## 結語：建國不是結束，是開始

很多人以為立國就破關了，但其實立國之後才是真正的挑戰。

你要管理外交、經濟、領主關係、軍事部署……一個不小心，你辛辛苦苦建立的王國就可能土崩瓦解。

但這就是騎砍2最好玩的地方——**你不是在玩一個遊戲，你是在活一段人生。**

祝你好運，未來的國王/女王陛下。🗡️
