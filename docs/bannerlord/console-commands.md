# 騎砍2 | 密技與控制台指令：當你需要一點小幫助

---

## 前言

有些時候你就是想爽一把，或是測試某些東西。騎砍2內建控制台指令可以做到大部分事情。但——

> ⚠️ **警告：開控制台 = Steam 成就永久鎖定。** 一旦用過密技，這個存檔再也拿不到任何成就。而且第一輪通關前強烈不建議用，會徹底毀掉遊戲體驗。

建議開一個獨立存檔玩密技，不要汙染你的主力存檔。

---

## 一、開啟控制台

### 步驟

1. 在 Steam 收藏庫 → 騎砍2 → 右鍵 → 內容
2. 啟動選項 → 輸入：`-console`
3. 啟動遊戲
4. 遊戲中按 `Alt + ~` 開啟控制台

---

## 二、常用指令

### 金錢與資源

| 指令 | 效果 |
|------|------|
| `campaign.add_gold_to_hero [數字]` | 加錢 |
| `campaign.add_attribute_points_to_hero [數字]` | 加屬性點 |
| `campaign.add_focus_points_to_hero [數字]` | 加專精點 |
| `campaign.add_renown_to_clan [數字]` | 加家族聲望 |
| `campaign.add_influence [數字]` | 加影響力 |

### 物品與裝備

| 指令 | 效果 |
|------|------|
| `campaign.give_item_to_main_party [物品ID]` | 獲得物品 |
| `crafting.unlock_all_parts` | 解鎖全部鍛造配件 |

### 部隊與戰鬥

| 指令 | 效果 |
|------|------|
| `campaign.give_troops [兵種ID] [數量]` | 獲得士兵 |
| `campaign.add_companion` | 招募隨機夥伴 |
| `campaign.set_speed [數字]` | 設定地圖移速 |

### 地圖與情報

| 指令 | 效果 |
|------|------|
| `campaign.reveal_map` | 全地圖開圖 |
| `campaign.set_skill_main_hero [技能] [等級]` | 設定主角技能等級 |

## 四、注意事項

- 開控制台 = Steam 成就鎖定
- 有些指令可能導致存檔損壞（極少數）
- 建議開新存檔玩密技，不要汙染你的主力存檔
