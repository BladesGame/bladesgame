# 騎砍2 | 外交系統詳解：戰爭不是唯一選項

---

## 前言

原版騎砍2的外交很陽春——宣戰、停戰、沒了。但即便在這樣的限制下，懂得操控外交的人，可以用最小代價統一全大陸。

---

## 一、宣戰機制

### 1.1 為什麼會開戰

- 領主投票通過宣戰
- 邊境衝突升級（掠奪村莊、攻擊商隊）
- 勢力平衡打破（某國太弱就會被圍毆）

### 1.2 什麼時候該宣戰

| 時機 | 原因 |
|------|------|
| 目標國正在跟另一國打仗 | 雙線作戰，分身乏術 |
| 目標國剛打完大仗 | 兵力尚未恢復 |
| 目標國有領主不滿 | 打完可以挖角 |

### 1.3 什麼時候不該宣戰

- 你自己才剛打完仗，兵力還沒恢復
- 目標國是當前的霸主（你打不贏）
- 你跟其他國還有戰事（不要雙線）

---

## 二、停戰談判

### 2.1 停戰條件

發起停戰 → 跟對方國王見面 → 談判條件：

- **你佔優勢**：可以要求對方付戰爭賠款
- **平手**：雙方互不支付
- **你劣勢**：你必須付錢才能停戰

### 2.2 何時該停戰

| 情境 | 行動 |
|------|------|
| 你已達成戰爭目標（佔了想要的城） | 立刻停戰 |
| 對方軍團比你的大兩倍 | 花錢停戰，不要硬撐 |
| 有第三國趁亂宣你 | 立刻跟第一國停戰 |

---

## 三、影響力系統

### 3.1 影響力怎麼來

- 參加軍團打仗
- 捐贈俘虜給領主城鎮
- 捐贈士兵給領主
- 政策加成

### 3.2 影響力怎麼用

| 用途 | 消耗 |
|------|------|
| 創建軍團 | 依規模 |
| 提案政策 | 50-100 |
| 否決他人提案 | 依提案 |
| 提名封地歸屬 | 依影響力 |

> 💡 影響力就是政治資本。存著關鍵時刻用，不要隨便花。

---

## 四、裝 Diplomacy MOD 之後

原版外交太簡陋，裝了 Diplomacy MOD 後會加入：

- **互不侵犯條約**：雙方約定一段時間內不打
- **聯盟**：A 國打 B 國 = 你的盟國也會自動參戰
- **威脅**：要求對方割讓領地，不從就宣戰
- **附庸**：對方變成你的附庸國，付貢金
- **分封系統**：更複雜的領主關係

> 這 MOD 是自立為王玩家的標配。詳見 [騎砍2 | 2026 必裝 MOD 推薦](./mod-recommendations)。

---

## 五、戰爭以外的手段

### 5.1 經濟壓制

- 掠奪敵國村莊 → 原料停產 → 城鎮收入下降
- 攻擊敵國商隊 → 斷他們財路
- 買下敵國城鎮的工廠 → 再賣掉（搞亂他們經濟）

### 5.2 政治滲透

- 挖角敵方領主 → 帶走封地 → 敵國變弱
- 支持敵國內部的叛軍 → 內部不穩
- 跟敵國的敵國結盟 → 形成包圍網
