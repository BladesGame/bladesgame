# 騎砍2 | 罪犯與海盜玩法：當個壞人也不錯

---

## 前言

正義太累？試試犯罪路線。掠奪村莊、走私貨物、襲擊商隊——壞事做盡，但賺錢超快。

---

## 一、犯罪玩法基礎

### 1.1 幹壞事的代價

| 行為 | 賺多少 | 扣什麼 |
|------|:--:|------|
| 掠奪村莊 | 2000-8000 金 | 聲譽 -、跟村莊/領主關係 - |
| 攻擊商隊 | 2000-5000 金 | 跟該國開戰 |
| 偷竊貨物 | 500-2000 金 | 城市犯罪率 + |
| 走私（Mod） | 不定 | 被抓到扣關係 |

### 1.2 聲譽的影響

聲譽太低：
- 領主不願意跟你打交道
- 城鎮可能拒絕你進入
- 但對犯罪玩法來說，這反而是一種身分標誌

---

## 二、犯罪路線

### 2.1 掠奪者

```
找沒有守軍保護的村莊 → 掠奪 →
賣戰利品 → 在被報復前換區域 →
重複
```

- 優點：收入高，來錢快
- 缺點：全村恨你，該國會派兵追殺你

### 2.2 走私者

需要對應 MOD（如 Fourberie）：

```
在低價區買違禁品 → 運到高價區賣 →
賄賂守衛避免檢查 →
利潤比正常貿易高 2-3 倍
```

### 2.3 海盜

需要對應 MOD：

```
在沿海區域活動 → 攻擊海上商船 →
搶貨物 + 俘虜 → 賣掉 →
高風險高回報
```

---

## 三、犯罪流生存技巧

1. **不要在同一個區域作案太久**：掠奪完就換國
2. **保持高移速**：全騎兵 + 少量貨物 = 跑得快
3. **賄賂守衛**：犯罪率高的城鎮可以賄賂進入
4. **躲到偏遠地區**：庫塞特東部、阿塞萊沙漠 = 沒人管

---

|------|------|
| Fourberie | 完整犯罪系統（偷竊、暗殺、走私、賭博） |
