# 騎砍2 | 2026 必裝 MOD 推薦與安裝教學

---

## 前言：原版很好玩，裝了 MOD 更好玩

先說一個殘酷的事實：騎砍2雖然是神作，但有些地方真的很讓人抓狂。

- 每次跟領主講話要等讀取畫面？
- 打完仗要一個一個手動升級士兵？
- 外交選項只有「宣戰」和「停戰」二選一？

這就是 MOD 存在的意義。

這篇精選 **必裝 MOD**，從生活品質改善到大幅擴充功能都有。

> ⚠️ **安裝 MOD 前請備份存檔！MOD 衝突可能導致壞檔！**

## 一、MOD 安裝基礎教學

### 1.1 兩種安裝方式

| 方式 | 優點 | 缺點 |
|------|------|------|
| **Steam 工作坊** | 一鍵訂閱、自動更新 | MOD 數量較少 |
| **Nexus Mods + Vortex** | MOD 最全、可手動管理版本 | 需要下載管理器和手動排序 |

> 🎯 新手推薦 Steam 工作坊。進階玩家推薦 Nexus Mods + Vortex。

### 1.2 四個必裝前置 MOD

無論你用什麼方式裝 MOD，這四個先裝（它們是 80% MOD 的依賴庫）：

| 順序 | MOD 名稱 | 作用 |
|:--:|------|------|
| 1 | **Harmony** | MOD 開發框架 |
| 2 | **ButterLib** | 擴展函數庫 |
| 3 | **UIExtenderEx** | UI 自訂擴展 |
| 4 | **Mod Configuration Menu (MCM)** | 遊戲內調整 MOD 設定 |

> ⚠️ **安裝順序不能亂！** Harmony → ButterLib → UIExtenderEx → MCM，照這個順序裝。

### 1.3 MOD 排序規則

在啟動器中，MOD 順序很重要：

```
[前置 MOD]         ← Harmony, ButterLib, UIExtenderEx, MCM
[大型系統 MOD]     ← Diplomacy, Realistic Battle Mod
[中型功能 MOD]     ← Improved Garrisons
[小型功能 MOD]     ← Distinguished Service
[畫質/UI MOD]     ← 放在最下面
```

## 四、常見問題

### Q：裝 MOD 後遊戲崩潰怎麼辦？

1. 檢查 MOD 版本是否對應遊戲版本
2. 檢查前置 MOD 是否都裝了（Harmony 等四個）
3. 檢查 MOD 排序（前置在最上面）
4. 逐一關閉最近安裝的 MOD 排查

### Q：裝 MOD 後存檔壞了？

懶人包的四個 MOD 都**相容現有存檔**。但大型 MOD（如 Banner Kings、Calradia Expanded）通常需要開新檔。

### Q：成就會被鎖嗎？

裝 MOD 後 Steam 成就會被禁用。如果在意成就，用 **Achievement Unblocker** MOD 解除限制。

## 結語：適量就好，別裝太多

MOD 是騎砍2長壽的祕密之一，但裝太多會有副作用：

- 開遊戲要五分鐘
- 隨機崩潰（而且不知道是哪個 MOD 的問題）
- 存檔肥到幾 GB

**建議：** 裝 5-10 個真正需要的就好。生活品質 MOD 優先，內容 MOD 選你最想要的 2-3 個。
