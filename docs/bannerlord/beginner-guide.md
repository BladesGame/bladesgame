# 騎砍2 | 新手求生指南：從被山賊追著跑到追著山賊跑

---

## 前言：你不是天選之人，你是個路人甲

先說清楚一件事：騎砍2不是無雙遊戲。你剛進遊戲的時候，**你就是個普通人**。你連三個山賊都打不贏，你的馬跑得比別人的驢還慢，你的錢包比乞丐還空。

但這正是騎砍2好玩的地方——**看著自己從被山賊追著滿地圖跑，變成帶著兩百精兵橫掃大陸，那種爽感是無可取代的。**

這篇指南就是幫你度過最痛苦的前期。

---

## 一、這遊戲到底在玩什麼？

《騎馬與砍殺2：霸主》是一款中世紀沙盒角色扮演遊戲。你扮演一個無名冒險者，在卡拉迪亞大陸上從零開始。

核心循環一句話：

```
打仗 → 賺錢 → 招更強的兵 → 打更大的仗 → 攻城 → 建國
  ↑_______________________________________________↓
```

- **自由度極高：** 你可以當傭兵、商人、封臣、國王，甚至純粹跑商一輩子
- **沒有主線強迫你：** 只有一個可選的龍旗任務，不做也能玩
- **目標自己訂：** 有人想統一全大陸，有人只想當個富商安享晚年

## 二、角色創建：出身與屬性怎麼選？

### 2.1 文化選擇

| 文化 | 特色加成 | 一句話評價 |
|------|----------|------|
| **帝國** | 駐軍薪資-20% | 什麼都會，最適合新手 |
| **庫塞特** | 騎兵地圖速度+10% | 公認後期最強，騎射很噁心 |
| **巴坦尼亞** | 森林移動+10% | 弓兵強到靠北，近戰是紙糊的 |
| **瓦蘭迪亞** | 戰鬥經驗+20% | 騎兵衝鋒很爽，弩兵陣地戰無敵 |
| **斯特吉亞** | 雪地移動+10% | 步兵硬得像坦克，移動像烏龜 |
| **阿塞萊** | 沙漠移動+10%、貿易懲罰-10% | 騎兵便宜好用 |

> 🎯 **新手推薦：帝國。** 地理位置在地圖中央，兵種最均衡，駐軍減免讓你養兵壓力小。

### 2.2 出身選擇

創角時會被問幾個背景問題，影響起始屬性：

| 出身 | 推薦度 | 原因 |
|------|:--:|------|
| **商人** | ⭐⭐⭐ | 起始資金多、交易技能高 |
| 傭兵 | ⭐⭐ | 戰鬥技能好、起始裝備佳 |
| 貴族 | ⭐⭐ | 騎術和魅力高 |

> 🎯 **新手推薦：商人出身。** 起始資金2000金，有錢就能僱人幫你打。

### 2.3 屬性點法

| 屬性 | 點數 | 原因 |
|------|:--:|------|
| 活力 | 2 | 影響單手/雙手/長桿，戰鬥基礎 |
| 耐力 | 2 | 影響騎術和鍛造 |
| 社交 | 2 | 交易、魅力、領導力——決定你能帶多少兵、賺多少錢 |
| 智力 | 2 | 管理學影響帶兵上限 |

> ⚠️ 控制力（遠程）前期不用點。弓和弩前期傷害低，中期有裝備再回頭補。

### 2.4 技能重點

1. **單手+盾牌**：容錯最高，新手首選
2. **騎術**：點到能騎戰馬就夠（60）
3. **領導力**：決定帶兵上限，必點
4. **交易**：初期賺錢靠它
5. **鍛造**：後期印鈔機，前期不急

## 三、前五小時你該做什麼？

### 3.1 別急著打仗，先去競技場

每個城市都有競技場，進去打練習賽。**這是前期最安全的練功方式**：

- 不會真的死掉（輸了只是昏倒）
- 可以練武器熟練度
- 贏了有獎金和裝備
- 帝國競技場裝備統一（單手劍+盾），最容易贏

> 💡 連續泡競技場幾輪，輕鬆賺到 3000-5000 金。

### 3.2 接村莊任務賺第一桶金

去各村莊找村長（站在村子裡的那個大叔），接任務：

| 任務 | 報酬 | 難度 |
|------|------|:--:|
| 送貨 | 300-800 金 | 簡單 |
| 找失蹤村民 | 200-500 金 | 簡單 |
| 打土匪 | 1000-3000 金 | 需要能打 |

> 送貨任務最穩，前期就是一直送貨、存錢。目標存到 5000 金。

### 3.3 招兵買馬，但不要貪心

有了錢第一件事：**去村莊招兵**。

新手常犯的錯誤是招太多兵養不起：

- **初期 15-20 人夠用了**
- 士兵的薪水是持續支出
- 帝國新兵 → 帝國步兵，這條線最穩定

### 3.4 打山賊練等

當你有 15-20 個兵之後，開始找地圖上的劫匪（人數少於你的隊伍）。

打山賊的目的不是賺錢，是：
1. 練你自己的戰鬥技巧
2. 練兵的等級
3. 撿裝備賣錢
4. 累積聲望（聲望 = 家族等級）

## 四、雇傭兵：最被低估的戰略武器 🧠

很多人以為雇傭兵只是「過渡階段」，賺點錢就要趕快當封臣。大錯特錯！

**雇傭兵是整個遊戲最靈活的身分。**

### 雇傭兵 vs 封臣，差在哪？

| | 雇傭兵 | 封臣 |
|------|------|------|
| 加入門檻 | 幾乎無 | 家族等級2+、國王同意 |
| 退出代價 | 零，隨時走人 | 扣關係、可能被追殺 |
| 敵對關係 | **不會**跟任何人變敵人 | 自動跟敵國所有人敵對 |
| 收入 | 日薪+戰利品 | 封地稅收+戰利品 |
| 機動性 | 今天打A國，明天幫A國打B國 | 綁定一國，不能隨便跳 |

### 核心戰略：用雇傭兵身分操控大陸局勢

這遊戲有一個隱藏機制：**你幫誰打仗，誰就會變強；你打誰，誰就會變弱。**

而雇傭兵的身分，讓你可以自由操控這個天秤：

1. **觀察地圖**：打開百科全書（N鍵），看各國城池數量和兵力
2. **找出最強的國家**：假設庫塞特已經吃掉半個帝國
3. **加入庫塞特的敵人**：當阿塞萊的雇傭兵，專打庫塞特
4. **削弱庫塞特**：抓他們領主、打他們軍團、圍他們的城
5. **見好就收**：庫塞特變弱了 → 退出 → 看看下一家誰太強
6. **換邊再來**：這次加入瓦蘭迪亞，去打另一個膨脹的國家

> 💡 **一句話總結：當雇傭兵的終極目標不是賺錢，是確保卡拉迪亞沒有任何一國能獨大。**

這樣等到你自立為王的時候，面對的是一個「各國互相制衡、誰都無法碾壓你」的大陸，而不是某個已經吃掉半張地圖的怪物。

## 五、這些坑千萬別踩 🕳️

| # | 錯誤 | 後果 | 正確做法 |
|:--:|------|------|------|
| 1 | 太早當封臣 | 被敵對國領主追著打、無法脫身 | 先當雇傭兵就好，封臣等家族等級3再說 |
| 2 | 太早攻城 | 全軍覆沒 | 沒有 100+ 精兵不要想攻城 |
| 3 | 錢全買裝備 | 沒錢養兵，兵跑光 | 永遠留至少養兵 7 天的薪水 |
| 4 | 無視俘虜系統 | 錯過穩定收入 | 俘虜賣給酒館贖金經紀人 |
| 5 | 忽視食物多樣性 | 士氣低 → 戰鬥力降 | 至少 3-4 種不同食物 |
| 6 | 同時跟多國開戰 | 被圍毆到亡國 | 一次只打一個敵人 |

## 六、快速上手懶人包

如果你是那種「字太多了」的人：

1. 創角：**帝國 + 商人出身**
2. 前 3 天：打競技場賺 5000 金 → 招 15 個帝國新兵
3. 第 4-7 天：打劫匪練兵 + 接送貨任務
4. 第 2 週：有 30 兵 + 10000 金 → 當雇傭兵，開始操控各國勢力平衡
5. 之後：持續當雇傭兵牽制強國 → 存錢買工廠 → 等各國互相削弱後再自立
