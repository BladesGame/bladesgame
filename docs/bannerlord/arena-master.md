# 騎砍2 | 競技場大師指南：從賭徒到冠軍

---

## 前言

競技場不只是新手練功的地方。隱藏獎勵、特殊對手、高難度挑戰——真正的競技場大師可以靠這個發財。

---

## 一、競技場基礎

### 1.1 三種模式

| 模式 | 說明 | 風險 |
|------|------|------|
| 練習賽 | 免費參加，練技能 | 無 |
| 錦標賽 | 淘汰賽，贏了有好獎品 | 輸了淘汰 |
| 下注自己 | 賭自己贏 | 輸了賠錢 |

### 1.2 各文化競技場特色

| 文化 | 典型裝備 | 難度 |
|------|------|:--:|
| 帝國 | 單手劍+盾 | 簡單 |
| 巴坦尼亞 | 弓+單手 | 中等 |
| 瓦蘭迪亞 | 騎槍+單手 | 中等 |
| 庫塞特 | 弓騎 | 難 |
| 斯特吉亞 | 斧+盾 | 簡單 |
| 阿塞萊 | 標槍+騎兵 | 難 |

> 💡 新手優先泡帝國競技場，裝備統一、公平、容易贏。

---

## 二、競技場賺錢

### 2.1 錦標賽獎品

贏得錦標賽可以獲得隨機獎品：
- 武器（通常是中高階）
- 盔甲
- 馬匹（最值錢）
- 旗幟

> 好馬可以賣到 5000-15000 金。

### 2.2 下注策略

- 前期你不強 → 不要賭
- 中期你能穩定打贏 → 每場下注自己
- 錦標賽下注倍率高，連贏幾場賺翻

---

## 三、各文化競技場打法

### 帝國競技場

```
裝備：單手劍+盾
打法：舉盾繞圈 → 對手砍空 → 橫砍 → 後退 → 重複
最簡單，練基本功就來這裡。
```

### 巴坦尼亞競技場

```
裝備：弓+箭+單手
打法：先射箭，箭射完再近戰
注意站位，不要被圍毆。
```

### 瓦蘭迪亞競技場

```
裝備：騎槍+單手劍+盾
打法：騎馬衝鋒刺擊 → 繞圈 → 再衝
騎術不熟的話很難打。
```

### 庫塞特競技場

```
裝備：弓+箭+騎馬
打法：邊騎邊射，全場風箏
最難的競技場，新手不要碰。
```

---

## 四、隱藏獎勵

有些競技場有特殊錦標賽，贏了可以獲得：
- 獨特武器（數值比一般武器高）
- 稀有旗幟
- 大量聲望

> 看到錦標賽一定要參加。
