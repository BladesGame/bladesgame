# 騎砍2 | 戰鬥技巧完全指南：從被砍到砍人

---

## 前言

騎砍不是無雙割草，戰鬥系統有四向攻擊、格擋、踢擊、騎乘加成……搞懂這些，你就不再是那個被山賊追著跑的路人甲。

---

## 一、基礎操作：四向攻擊與格擋

### 1.1 攻擊方向

| 操作 | 效果 | 使用場景 |
|------|------|------|
| 滑鼠向上 + 左鍵 | 上劈 | 傷害最高，對付無盾敵人 |
| 滑鼠向下 + 左鍵 | 下砍 | 打腳、破盾後補刀 |
| 滑鼠向左/右 + 左鍵 | 橫砍 | 範圍最大，騎馬首選 |
| 滑鼠移動 + X 鍵 | 刺擊 | 長矛/長桿用，騎馬傷害×3 |

### 1.2 格擋

- **有盾牌**：舉右鍵自動格擋（但會消耗盾牌耐久）
- **無盾牌**：右鍵 + 滑鼠方向手動格擋（高手向）
- **踢擊（E鍵）**：破防神技！對面舉盾 → 踢開 → 立刻補一刀

### 1.3 騎乘戰鬥

- 騎馬時武器傷害 = 基礎傷害 × 速度倍率
- 高速衝鋒一刀可打出 2-3 倍傷害
- 長桿武器騎乘時會自動架矛

## 二、單挑三招

### 2.1 繞圈破盾流（新手最強）

```
全程舉盾 → 繞對手逆時針轉圈 → 等對手砍空 →
立刻回一刀橫砍 → 後退 → 重複
```

### 2.2 踢擊破防流

```
對手舉盾 → E鍵踢開 → 補上劈或刺擊 → 後退 → 重複
```

### 2.3 騎馬單挑

```
繞對方騎兵逆時針 → 從左側超車 → 一刀砍馬 →
馬倒下人摔落 → 補刀收割
```

## 三、野戰指揮

### 3.1 快捷鍵

| 鍵 | 功能 |
|:--:|------|
| F1 | 移動/位置指令 |
| F2 | 朝向 |
| F3 | 陣型 |
| F4 | 開火/停火 |
| F6 | 交給你指揮（AI 自動） |

### 3.2 錘砧戰術

最經典的戰術：步兵扛線（砧）+ 騎兵兩面包抄（錘）。

1. 步兵 F3+F2 開盾牆面向敵方
2. 弓手放在步兵後方高處
3. 騎兵分兩隊放側翼
4. 敵方接觸步兵 → 騎兵 F1+F3 繞背衝鋒

### 3.3 弓騎風箏流

```
弓騎 F1+F4 散開 → F4 自由射擊 →
敵方追弓騎（陣型拉散）→ 你帶重騎兵衝鋒敵方後排 →
步兵推進收尾
```

### 3.4 地形利用

| 地形 | 效果 |
|------|------|
| 高地 | 弓兵射程+傷害 +20% |
| 河流/橋 | 窄口堵路，10 個步兵扛 50 人 |
| 森林 | 騎兵衝鋒速度 -50%，步兵主場 |

## 四、攻城戰

### 4.1 進攻

1. 造投石機 → 優先打城牆和敵方投石車
2. 步兵推攻城塔 + 破門錘
3. 弓兵在城下壓制城牆守軍
4. 塔靠牆 → **不要第一個上！**
5. 城牆戰舉盾推進，先清弓兵再打步兵

### 4.2 防守

- 投石車優先打攻城塔和破門錘
- 弓兵放城牆射推塔的敵兵
- 最強步兵放城門後準備巷戰
