# 騎砍2 | 步兵陣地戰完全指南：扛住就是勝利

---

## 前言

步兵是每支軍隊的脊樑。沒有步兵扛線，你的弓兵會被騎兵輾過去，你的騎兵衝鋒完沒人補位。這篇講怎麼讓你的步兵變成一座移動要塞。

---

## 一、步兵的定位

| 角色 | 說明 |
|------|------|
| 扛線 | 擋住敵方步兵和騎兵衝鋒 |
| 保護 | 站在弓兵和弩手前面 |
| 推進 | 攻城時推塔、破門、巷戰 |
| 補位 | 騎兵衝完拉走後，步兵填上去 |

## 三、陣型與戰術

### 3.1 盾牆陣（F3+F2）

最常用的防禦陣型。步兵舉盾靠攏：

- 正面防禦力極高
- 騎兵衝不過去
- 缺點：移動很慢、側面和背後脆弱

### 3.2 方陣（F3+F3）

對抗騎兵的最佳陣型。四面都有盾牌：

- 騎兵從哪個方向衝都會撞上盾牌
- 但弓兵能射到你
- 只在被騎兵包圍時用

### 3.3 戰術流程

1. 步兵 F3+F2 排盾牆在最前面
2. 弓兵在步兵後方高處
3. 騎兵在側翼待命
4. 敵方接近 → 騎兵側面衝鋒
5. 騎兵拉走 → 步兵推進收割

---

## 四、反騎兵技巧

騎兵衝鋒是步兵最大的威脅。應對方法：

1. **長矛/長桿武器**：瓦蘭迪亞長矛兵專門反騎
2. **盾牆陣**：騎兵衝不穿密集盾牆
3. **地形**：森林、橋樑、狹窄地形 = 騎兵廢掉
4. **圍毆**：騎兵停下來後讓步兵圍上去
