# 關於 Blades Game 游刃攻略

## 這是什麼？

**Blades Game 游刃攻略** 是一個繁體中文遊戲攻略網站，專注於深度、原創、結構化的遊戲指南。

我們跟一般遊戲攻略站的差別：

- 🎯 **深度而非廣度：** 一篇攻略從頭講到尾，而不是幾句零散心得
- 📊 **數據驅動：** 兵種對比、收益分析，全部有數字支撐
- 📍 **具體地名：** 每個村莊、每個城鎮、每個招募點都標清楚
- 🔄 **持續更新：** 跟隨遊戲版本更新，不讓攻略過時

## 為什麼叫「遊刃」？

取自成語「遊刃有餘」—— 玩遊戲就該輕鬆寫意，一刀到位。

不囉嗦、不搬運、不抄襲。每一篇都是親自遊玩後的原創心得。
