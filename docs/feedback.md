# 攻略內容回報

## 發現攻略有誤？我們想聽你說

《騎馬與砍殺2：霸主》是一款機制複雜的遊戲，同一件事可能因為版本、平台、Mod 而結果不同。我們盡力確保每篇攻略準確，但仍可能有疏漏。

如果你發現以下任何問題，歡迎回報：

- 攻略內容與遊戲實際情況不符
- 版本過時（遊戲更新後機制已改）
- 數據錯誤（價格、傷害值等）
- 排版或連結問題
- 想許願新攻略主題

---

## 回報格式

為了讓我們能快速確認問題，請盡量提供以下資訊：

| 欄位 | 說明 | 範例 |
|------|------|------|
| **文章名稱** | 哪篇攻略有問題？ | 新手求生指南 |
| **問題段落** | 哪一段、哪個表格？ | 第三章 3.2 節，競技場收入寫 500 金 |
| **實際情況** | 你在遊戲中遇到的狀況 | 我打贏帝國競技場只有 250 金 |
| **遊戲版本** | 你的遊戲版本號 | v1.4.2 |
| **平台** | Steam / Epic / Game Pass | Steam |
| **是否裝 Mod** | 有/無（有的話列幾個主要的） | 無 |

> 📝 如果能附上截圖，對我們查證會有很大幫助。

---

## 回報方式

來信至：**contact@bladesgame.com**

主旨請寫：`[攻略回報] 文章名稱 - 問題簡述`。

---

## 回報後會發生什麼事？

1. **我們收到回報** → 在 3-7 天內查證
2. **確認問題** → 比對遊戲實測或官方更新說明
3. **修正文章** → 更新攻略內容
4. **文章底部更新紀錄** → 會註明修正了什麼，並感謝回報者

你可以在每篇文章最下方的「更新紀錄」看到修改歷史。
